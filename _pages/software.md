Software
------

The goal of the following software packages is primarily to make our algorithms more accessible to the public.

### SteinerPy
[**Software package**](https://github.com/berendmarkhorst/SteinerPy/tree/main) for solving Steiner tree and forest optimization problems with an exact and open-source solver.

### APRpy  
[**Software package**](https://github.com/berendmarkhorst/APRpy) for benchmarking automated pipe routing problems.

### BRAIN
[**Dashboard**](https://brainopt.streamlit.app) that enables practitioners and researchers to find a suitable base station for a Mobile Stroke Unit.

### SJORS
**Software for problem-based scenario generation in stochastic programming problems** (soon online).
